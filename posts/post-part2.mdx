# MPV 2 日目

どうも Lua というプログラミング言語で制御するのがいいとわかった。

```
$HOME/.config/mpv/scripts/
```

以下に、ES5 の Javascript か Lua のソースをいれると自動で読み込んでくれるみたいだ。
といっても、どっちもわかんないし…。
ブラウザを開かなくても youtube を検索して、ちょっとした小窓にサムネイルと動画タイトルが並んで
クリックしたら再生されるような、そーんな機能が欲しいなぁ。
そんなプログラミング力はないから検索しまくろっと…。

ついでにこれまたよくわからない呪文みたいなのも追加しました。
osc.conf

```
layout=bottombar
vidscale=no
scalewindowed=1.75
scalefullscreen=1.75
boxalpha=160
hidetimeout=2000
```

なんでこれが input.conf と別ファイルになるのか、疑問は深まるばかり。
思い通りに設定を書けるようになるには、動画ソフトの技術的な事を勉強してからじゃないと駄目なんだろうな。
mpv.io のリファレンスの翻訳を欲するより先に、そういった事前知識が必要って今から挫折しそう。
でもこのソフト以外にもそういうの多そうだから、一つ覚えていけば活路はあるのだろうか。

とにかく不便を享受しながら、耐えて覚えるよう頑張ってみよう。
